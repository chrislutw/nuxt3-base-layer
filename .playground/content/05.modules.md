## Robots

這個模組提供了 robots.txt 的功能

限制了除正式環境外的 robots.txt 為 `User-agent: * Disallow: /`

這樣可以避免搜尋引擎索引這些環境。

### 增加 sitemap 設定

只要在 `runtimeConfig.public.siteUrl` 設定網站網址，就會自動在 robots.txt 中加入 sitemap 設定。

### 自訂 robots handler

可以在 `baseLayerRobots.customHandler` 設定自訂的 handler 路徑

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  baseLayerRobots: {
    customHandler: '/path/to/custom/robots.handler.ts'
  }
})
```
